# Dedinsky for Judge

Campaign website for Paul Dedinsky, candidate for Waukesha County Circuit Court Judge.

## About

Static campaign site with plain HTML, CSS, and vanilla JS. No build step, no framework, no dependencies beyond Google Fonts.

## Structure

```
├── index.html              Landing page
├── about.html              Bio, career, education, community involvement
├── vote.html               Election info and judicial philosophy
├── endorsements.html       Endorsements
├── support.html            Contact and get involved
├── donate.html             Contribution page
├── css/
│   └── style.css           Shared styles and responsive breakpoints
├── js/
│   └── nav.js              Mobile menu toggle (only JavaScript on the site)
└── img/                    Photos and logo assets
```

## Deployment

Hosted on [Cloudflare Pages](https://pages.cloudflare.com/). No build command, no output directory, Cloudflare serves the repo root as-is.

## Local Preview

Open `index.html` in a browser. Everything works without a server.
